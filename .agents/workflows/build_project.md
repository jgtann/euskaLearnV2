# Build Project Workflow

This workflow documents the standard procedure for installing dependencies and building the Euskal Sustatzailea (NextJS) project.

## Steps

1. **Install Dependencies:**
   Run the following command to install all necessary packages from `package.json`:
   ```bash
   npm install
   ```
2. **Build the Project:**
   Once dependencies are installed, you can create a production build by running:
   ```bash
   npm run build
   ```
   *Note: This command will compile the application into the `.next` directory.*
